Place your generated hero animation files here.

Primary file names used by the site:
- `hero-animated.mp4` (preferred)
- `hero-animated.webm` (optional secondary source)
- `hero-animated.avif` or `hero-animated.gif` (fallback when video fails or reduced-motion preference is active)
- `hero-animated-poster.jpg` (optional poster image while video loads)

Recommended export settings:
- 1920x1080 or 1600x900
- 5 to 7 seconds seamless loop
- H.264 MP4 (`hero-animated.mp4`)
- Target size <= 1.5 MB (soft limit)
- Keep subject centered and avoid embedding text in the animation
